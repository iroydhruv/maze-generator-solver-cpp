\# Maze Generator \& Solver (C++)



A console-based Maze Generator and Solver developed using Object-Oriented Programming principles in C++.



\## Features



\- Random Maze Generation

\- DFS Maze Solver

\- BFS Maze Solver

\- SQLite Database Integration

\- Save and Load Mazes

\- Modular OOP Design



\## Technologies



\- C++

\- SQLite3

\- CMake

\- OOP
